# **App Name**: Desencontro CRM

## Core Features:

- Client Management: Enable the creation, editing, and deletion of client profiles.
- Interaction Tracking: Record and manage all interactions with clients, including notes and summaries. Use tool to suggest interaction prompts or next steps, improving follow-through based on successful actions with previous customers.
- Negotiation Management: Track the progress of negotiations and partnerships, managing key details such as status and agreements.
- Dashboard Visualization: Display key metrics and summaries in a dashboard for quick insights into client relationships and event progress.
- Lead Filtering: Implement search and filtering to quickly find and categorize leads based on their status.
- Custom Tags: Allow users to create custom tags for clients to categorize them and easily find certain attributes of their contacts.

## Style Guidelines:

- Primary color: Deep red (#8B0000) taken from the Instagram feed to evoke trust and professionalism.
- Secondary color: Gold (#FFD700) to represent the main color used in the brand.
- Body text: 'PT Sans', a humanist sans-serif, offering both a modern look and warmth, used for all standard text.
- Headline font: 'Space Grotesk' a proportional sans-serif with a computerized feel for impactful titles and headlines
- Use modern and professional icons to represent different actions and entities.
- Design a clean and structured layout to allow users to manage clients, leads, conversations, and negotiations efficiently.
